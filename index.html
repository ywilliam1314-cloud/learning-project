XYZ to JSON Studio

支持拖入单个 .xyz 文件，也支持拖入整个文件夹并自动递归转换。

Local Converter

把 .xyz 文件或文件夹直接拖进来

单文件模式会在右侧直接显示转换后的 JSON。文件夹模式会自动递归寻找其中的 .xyz 文件，调用本地 Flask 后端完成转换，并把对应的 .json 写回到每个 .xyz 所在的原目录。

文件夹写回依赖 Chromium 系浏览器的 File System Access 能力。推荐通过 http://127.0.0.1:2100 打开本页面。

如果你是直接双击 HTML 文件打开页面，浏览器无法自动连接本地后端，也无法可靠地把结果写回原文件夹。 请先启动 xyz_to_json.py 或 start_xyz_json_app.bat。
