# 随机强密码生成器

一个基于 **Python Flask + HTML** 的强密码生成 Web 应用，支持自定义长度、字符类型，以及一键生成、一键复制。

---

## 项目背景与开发历史

### 第一版需求（初始提示）

> 利用 Python 语言，写一个随机生成 10 位强密码的小程序，密码包含数字、小写字母、大写字母和特殊字符。使用 HTML 写一个页面，页面上输出生成的密码，有一个重新生成的按钮，按下后会重新生成。

**技术选型**：Flask（Python 后端）+ Jinja2 模板（HTML 前端），通过 REST API 实现前后端交互。

## 功能说明

### 密码生成规则（`app.py` → `generate_password()`）

- 长度：**用户自选，1–50 位**（默认 10 位）
- 字符集（均可独立开关）：
  - 数字：`string.digits`（0–9）
  - 小写字母：`string.ascii_lowercase`（a–z）
  - 大写字母：`string.ascii_uppercase`（A–Z）
  - 特殊字符：`!@#$%^&*()-_=+[]{}|;:,.<>?`
- 生成策略：当长度 ≥ 已启用类型数时，**先从每个启用类型各取 1 个**，保证各类型必然出现；再从合并字符集随机补足剩余位数；最后用 `random.shuffle()` 打乱顺序

### API 路由

| 路由 | 方法 | 参数 | 说明 |
|------|------|------|------|
| `/` | GET | 无 | 渲染首页，服务端预生成一个密码注入模板 |
| `/generate` | GET | `length`、`digits`、`lower`、`upper`、`special` | 返回 JSON `{"password": "..."}` 供前端 Fetch 调用 |

**`/generate` 参数说明：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `length` | int | 10 | 密码长度，范围 1–50 |
| `digits` | "0"/"1" | "1" | 是否包含数字 |
| `lower` | "0"/"1" | "1" | 是否包含小写字母 |
| `upper` | "0"/"1" | "1" | 是否包含大写字母 |
| `special` | "0"/"1" | "1" | 是否包含特殊字符 |

### 前端功能（`templates/index.html`）

- 页面加载时显示服务端预生成的密码（Jinja2 `{{ password }}`）
- **密码长度滑块**：范围 1–50，实时显示当前位数
- **字符类型开关**：数字 / 小写字母 / 大写字母 / 特殊字符，点击切换启用/禁用；全部取消选中时生成按钮自动禁用并提示警告
- **重新生成按钮**：读取当前配置，调用 `/generate` API，带 200ms 淡出/淡入动画切换密码
- **点击复制**：调用 `navigator.clipboard.writeText()`，成功后提示"已复制"，2 秒后恢复
- **动态强度条**：综合已选类型数（0–4）与密码长度计算强度，分"弱 / 中等 / 强 / 非常强"四档，配色实时变化
- UI 风格：深色毛玻璃卡片，蓝色主色调
